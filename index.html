Nova

Nova is a JavaScript framework based on XumaK's Danta. It provides a mechanism where the business logic is constructed as modular, granular and reusable pieces called Context Processors, has a simple and clear API which is independent from any particular technology and allows to easily test the logic either piece by piece or several pieces in conjunction.

  • Nova Core
  • Nova Server
  • Nova Firebase

Nova Core contains the framework engine. Can be used by itself or in conjunction with Nova Server and Nova Firebase.

Nova Server is a lightweight web server using Expressjs that enables building web APIs with Nova.

Nova Firebase is a lightweight wrapper that enables the use of the Nova framework in Firebase Functions.
